Stream 1: What is a variable?

A variable is a container that stores data, just like how tupperware stores leftovers

Variable Declarations

Var - old skool that is function and global scoped

let - block scoped and can be reassigned, like a nickname

const - block scoped and can't be reassigned, like a legal name (paying to change your name doesn't)
